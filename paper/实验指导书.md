### 2.2 统计显著性检验

**目标**：证明 EATA 的优势具有统计显著性

**方法**：

1. 对每支股票计算各方法的年化收益
2. 使用 paired t-test 比较 EATA 与每个基线
3. 报告 p-value，标注显著性水平

**Python 代码**：

```python
from scipy import stats
import numpy as np

def significance_test(eata_returns, baseline_returns):
    """
    eata_returns: list of 31 floats (每支股票的年化收益)
    baseline_returns: list of 31 floats
    """
    t_stat, p_value = stats.ttest_rel(eata_returns, baseline_returns)
    significance = ""
    if p_value < 0.01:
        significance = "**"
    elif p_value < 0.05:
        significance = "*"
    return t_stat, p_value, significance
```

**输出**：在 Table 2 中标注 $^*$ (p<0.05) 和 $^{**}$ (p<0.01)

**修改回测代码**：

```python
def backtest_with_cost(signals, prices, cost_rate=0.001):
    """
    cost_rate: 单边交易成本
    """
    portfolio_value = 1.0
    position = 0
  
    for t in range(1, len(signals)):
        if signals[t] != position:
            # 交易发生，扣除成本
            portfolio_value *= (1 - cost_rate)
            position = signals[t]
      
        # 计算收益
        if position == 1:
            portfolio_value *= (1 + (prices[t] - prices[t-1]) / prices[t-1])
  
    return portfolio_value
```

```python
import matplotlib.pyplot as plt
import numpy as np

def plot_cumulative_returns(results_dict, ticker='AAPL'):
    """
    results_dict: {'EATA': returns_array, 'Buy&Hold': returns_array, ...}
    """
    plt.figure(figsize=(10, 6))
```

```python
def analyze_by_market_regime(returns, market_returns):
    """
    按市场状态分析策略表现
    """
    regimes = []
    for i in range(20, len(market_returns)):
        period_return = (market_returns[i] / market_returns[i-20]) - 1
        if period_return > 0.05:
            regimes.append('Bull')
        elif period_return < -0.05:
            regimes.append('Bear')
        else:
            regimes.append('Sideways')
  
    # 按状态分组计算收益
    results = {}
    for regime in ['Bull', 'Bear', 'Sideways']:
        mask = [r == regime for r in regimes]
        results[regime] = np.mean(returns[20:][mask])
  
    return results
```

### 6.2 表达式稳定性分析

**目标**：分析不同时间窗口发现的表达式的一致性

**Python 代码**：

```python
def analyze_expression_stability(expressions_by_window):
    """
    分析不同窗口发现的表达式的相似度
    """
    from difflib import SequenceMatcher
  
    similarities = []
    for i in range(1, len(expressions_by_window)):
        expr1 = expressions_by_window[i-1]
        expr2 = expressions_by_window[i]
        sim = SequenceMatcher(None, expr1, expr2).ratio()
        similarities.append(sim)
  
    return np.mean(similarities), np.std(similarities)
```

```python
import time

def benchmark_inference_time(model, data, n_runs=100):
    """
    测量推理时间
    """
    times = []
    for _ in range(n_runs):
        start = time.time()
        _ = model.predict(data)
        times.append(time.time() - start)
  
    return np.mean(times) * 1000, np.std(times) * 1000  # 毫秒
```

## 十、常见问题与解决方案

### Q1: 实验结果不稳定怎么办？

**解决方案**：

1. 增加随机种子数量（建议 5 次）
2. 报告均值 ± 标准差
3. 使用 Wilcoxon 符号秩检验代替 t-test

### Q2: EATA 在某些股票上表现很差怎么办？

**解决方案**：

1. 诚实报告失败案例
2. 分析失败原因
3. 讨论方法的适用范围

### Q3: 训练时间太长怎么办？

**解决方案**：

1. 减少 MCTS episodes (K=400)
2. 使用更小的 buffer size
3. 并行训练多支股票

### Q4: 如何选择 31 支股票？

**建议标准**：

1. 覆盖不同行业（科技、金融、消费、医疗等）
2. 覆盖不同市值（大盘、中盘、小盘）
3. 覆盖不同波动率水平
4. 确保数据完整性（无长期停牌）
